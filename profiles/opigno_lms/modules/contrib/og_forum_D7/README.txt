OG Forum D7
http://drupal.org/project/og_forum_d7
=======================================

DESCRIPTION
-----------
This module provides an integration of Organic Groups
into the core Forum module.

Its working is mainly based on the node access system already provided by OG.
It add a layer of access on taxonomy system with the hook_query_TAG_alter().
It also optionally integrate with forum_access to provide moderator access
and with og_context to enable the correct context depending
on the forum.

REQUIREMENTS
------------
Drupal 7.x

Core Forum module
Organic Groups
